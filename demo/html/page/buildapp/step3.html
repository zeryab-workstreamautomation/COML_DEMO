## Adding Navigation We will use eventhandlers on `div`s to control what is shown in the pages. We have now added code to our frame through the `script` tag. Notice that unlike normal html, we define variables on a `this` object. The [This](@coml-fns/this.html) object represents an instance of the COML page (in this case the Frame). This is useful when the same html page is reused many times - each reuse has its own this object. In COML eventhandlers (such as 'onclick') can refer to the this object, as shown.
