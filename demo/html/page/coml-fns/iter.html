## `iter(parentTag:string) : number;` Finds the current element's 0 based iteration in a parent with the tag 'parentTag'. The iteration is found from the point at which this function was called. `@param parentTag` The tag of the parent from which to find our iteration `@returns iteration number` The 0 based iteration number. ## Use `iter()` is useful in evaluating the iteration number, for example, for generating an index in a table in which each row is generated by the iteration of a Component. ## Examples
${this.iter('for')+1}
${this.fruit}
