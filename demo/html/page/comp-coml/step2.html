A tabbed container comprises a content area and a selector (Tabs) area. The Tabs area contains buttons that select content to be shown. Using the [Pages](@pages.html) control, we can define a simple container like this:

The COML code above defines a tab btn strip `.u-tabs-strip` with a single div. This defines the selector button. It has a custom attribute `asset2load` which defines the COML asset that should be loaded when selected. The selector button calls `this.switchTab()` on click. The method `this.switchTab()` reads the `asset2load` attribute from the clicked button, finds the `pages` Component using using [the get method](@coml-fns/get.html), and uses its `show()` function to display the page. It also sets the `u-btn-sel` class on the tab that was clicked, to mark is as selected. Note that this is a normal COML document. You can test it on its own - this makes it easy to test and debug a template during development.
